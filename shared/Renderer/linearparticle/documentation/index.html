LinearParticle 0.50 Documentation


LinearParticle is a 2D particle system library for Clanlib. LinearParticle is aimed to be easy-to-use and can be used to make various of visual effects or projectile shooting in games. To get basic idea what LinearParticle is able do for you, the best way might be to check the examples that downloadable from the website or come along with this documentation.
